# Bandit Level 0 -> Level 1

- **Platform**: OverTheWire
- **Wargame**: Bandit
- **Date**: 2025-12-12 23:44

## Overview

Bandit Level 0 is the first level of the OverTheWire Bandit wargame. The goal is to log in to the server and read the password for the next level (Bandit 1) from a file in the home directory.

### **Connection:** 
**Host:** `bandit.labs.overthewire.org`  
**Port:** `2220`  
**User:** `bandit0`  
**Password:** `bandit0`

## Steps

### 1. SSH into the server

<Terminal title="local" command="ssh bandit0@bandit.labs.overthewire.org -p 2220" output="(enter password: bandit0 when prompted)" />

### 2. List files in home directory

<Terminal title="bandit0@bandit" prompt="bandit0@bandit:~$ " command="ls" output="readme" />

### 3. Read the password for Level 1

<Terminal title="bandit0@bandit" prompt="bandit0@bandit:~$ " command="cat readme" output="(password for bandit1 - use it to log in as bandit1 for Level 1)" />

The contents of `readme` are the password for Bandit Level 1. Use it to continue with `ssh bandit1@bandit.labs.overthewire.org -p 2220`.

## Summary

- SSH with the given credentials; the password for the next level is in a file in the home directory.
- Basic commands: `ls` to list files, `cat` to read file contents.
